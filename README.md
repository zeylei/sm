## 练习方法

网页版练习地址：[点击直达](https://zeylei.github.io/xm/)

1. 在输入框输入闪动的汉字。

3. 第一个面板里的文字列表，会显示每个字的正确率，当所有文字正确率达到70%以上，且你的击键达到`目标击键`，程序会自动在列表里添加一个字给你练。

4. 点击右侧导航里的`设置`，设置你的`目标击键`，每一段的字数，要练习的文本，以及皮肤。有程序员需要的黑色皮肤呦！
